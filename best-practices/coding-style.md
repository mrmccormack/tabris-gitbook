
# Coding Style

## ![âœ”] 3.3 Start a Codeblock's Curly Braces in the Same Line

**TL;DR:** The opening curly braces of a code block should be in the same line of the opening statement.

### Code Example
```javascript
  // Do
  function someFunction() {
    // code block
  }

  // Avoid
  function someFunction()
  {
    // code block
  }
```
